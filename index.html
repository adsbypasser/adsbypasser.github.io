Feature

This user script helps you

  • skip ads' count-down or continue page
  • block pop-up windows

Lite edition removes image-hosting site support from Full edition. If you prefer to use other userscripts to deal with image-hosting sites, you can use the Lite edition.

Any feature request or bug report is welcome. (either in English, 中文 or 日本語) The source repository is on GitHub. You could use GitHub to report issues or send pull requests, if you have an account.

You can configure some function in this page, please see here if you need more information.
